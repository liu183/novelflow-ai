## 技能列表

**核心公式**：日常 + 反常 = 戏剧性

**何时使用**：

## 核心方法论

### 1. 冲突公式

```
日常元素 + 反常元素 = 戏剧性
```

**示例**：
- 日常：省钱买商品
- 反常：捡到的钱带着诅咒
- 戏剧性：必须花掉但会有后果

### 2. 三幕式结构

```
第一幕：建置（25%）
├─ 开场画面
├─ 主角介绍
├─ 触发事件
└─ 第一幕高潮

第二幕：对抗（50%）
├─ 试炼阶段
├─ 中点转折
└─ 一无所有时刻

第三幕：结局（25%）
├─ 最终对决准备
├─ 高潮爆发（抑制+意外+转折）
└─ 新平衡建立
```

### 3. 角色7要素

1. **重复行为**：第一印象/标志性动作
2. **反差细节**：外表vs内在
3. **成长轨迹**：开始vs结束
4. **独特观察**：世界观
5. **特殊天赋/怪癖**：独有特质
6. **真实缺陷**：致命弱点
7. **强烈动机**：驱动行为的核动力

### 4. 场景蓝图

```
场景基础 → 节拍设计 → 五感整合 → 镜头规划 → 悬念设置
```

### 5. 只呈现不叙述

❌ 错误：直接说情绪或状态
```
他感到愤怒。
这是一个恐怖的夜晚。
```

✅ 正确：用具体细节展示
```
他的拳头捏得发白，额头上青筋暴起。
月光照不进的巷子，远处传来不明的呜咽。
```

### 6. 节拍序列

```
尝试1 → 障碍 → 失败 →
尝试2 → 更大障碍 → 险胜 →
尝试3 → 致命障碍 → 生死对决 →
最终突破
```

### 7. 悬念设置三法

1. **让问题带领情节**：设计关键问题
2. **关键时刻停顿**：草蛇灰线
3. **"盲人摸象"**：控制信息释放节奏

### 8. 质量评分

```
结构分（25%）+ 人物分（25%）+ 情节分（25%）+ 文字分（25%）= 100分

及格线：70分
优秀线：85分
```

## 快速参考

### 冲突来源

| 类型 | 对象 | 示例 |
|------|------|------|
| 人与人 | 敌人、对手 | 竞争者、反派 |
| 人与社会 | 规则、体制 | 法律、道德 |

| 人与自己 | 内心挣扎 | 信念、缺陷 |

### 对话原则

- ✅ 符合人设
- ✅ 有潜台词
- ✅ 有目的
- ❌ 废话
- ❌ 自我介绍
- ❌ 说明性太强

### 画面感技巧

1. 用熟悉事物类比
2. 只呈现，不叙述
3. 学习镜头语言
4. 调动五感

### 高潮三工具

1. **抑制**：先打压形成势能
2. **意外**：再次加压挑战极限
3. **转折**：绝境中出现转机

```
你：/灵感采集器 一个总是捡到钱的女孩
AI：[提供3个方案]
你：选方案A，继续
AI：[继续深入]
```

不需要按固定顺序，哪里需要用哪里：

```
- 写到一半对话卡住 → /对话优化师
- 场景太平淡 → /场景渲染师
- 不知道怎么改 → /质检验收员
```

## 注意事项

### 这些技能的特点

1. **系统性**：基于完整的创作理论
2. **实用性**：提供具体方法和示例
3. **可操作**：每个技巧都能直接应用
4. **全方位**：覆盖创作全流程

### 如何获得最佳效果

1. **提供足够信息**：技能越了解你的需求，效果越好
2. **迭代优化**：可以反复调用同一个技能
3. **组合使用**：多个技能配合效果更佳
4. **保持上下文**：在同一次对话中持续使用

### 常见误区

❌ **期望一次完美**：创作需要反复打磨
❌ **跳过结构**：好的结构是好故事的基础
❌ **忽视人物**：人物弧光让故事有灵魂
❌ **只写不改**：修改是提升质量的关键

## 进阶技巧

### 1. 建立创作档案

为每个项目创建档案：
- 灵感素材
- 结构大纲
- 角色档案
- 章节规划
- 修改记录

### 2. 分阶段推进

不要试图一次性完成所有阶段：
- 第一阶段：完成灵感到结构
- 第二阶段：完成角色到情节
- 第三阶段：完成创作到修改

### 3. 定期诊断

每完成一部分就用**质检验收员**诊断：
- 及时发现问题
- 避免后期大改
- 持续提升质量

### 4. 建立检查清单

为每个阶段建立检查清单：
- 灵感：冲突清晰吗？
- 结构：比例合理吗？
- 角色：有7要素吗？
- 场景：有五感吗？
- 对话：有潜台词吗？

## 常见问题

### Q: 这些技能适合所有类型的小说吗？

A: 核心原理是通用的，但具体应用时需要根据类型调整。比如悬疑小说更注重悬念设置，言情小说更注重人物关系。

### Q: 必须按顺序使用吗？

A: 不必。这些技能是模块化的，可以根据需要灵活调用。但整体上，灵感→结构→角色→情节→内容→修改的创作顺序是推荐的。

### Q: 我已经有初稿了，还用这些技能吗？

A: 非常适合！可以用质检验收员全面诊断，然后用其他技能针对性改进。

### Q: 这些技能会替我写小说吗？

A: 不会。这些技能是助手，提供方法、建议和示例，最终创作还是要靠你自己。它们的作用是帮助你写得更好。

### Q: 如何判断我的作品质量？

A: 使用质检验收员的四维度评分系统。85分以上是优秀作品，70分以上是合格作品。
